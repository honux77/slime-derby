# 🎮 Slime Derby (슬라임 더비)

귀여운 슬라임들의 흥미진진한 경주 게임! 

## 🎯 게임 특징

- 🏁 **2~1000명의 플레이어** 경주 지원
- 🎨 **다채로운 슬라임** 8가지 컬러
- ✨ **특수 효과**
  - 🚀 BOOST - 순간 가속
  - 💥 TRIP - 넘어짐
  - 💤 Sleep - 졸음
- 👥 **관중석 응원** 메시지
- 📊 **상세한 결과 통계**
- 📥 **결과 이미지 다운로드**
- 🎵 **배경음악 & 효과음**

## 🎮 플레이 방법

1. 플레이어 수 선택 (2~1000명)
2. 목표 시간 설정 (5~60초)
3. 플레이어 이름 커스터마이징 (선택사항)
4. START 버튼 클릭!

## 🏆 결과 화면

- 1등 도착 순간 스크린샷
- 순위별 기록 및 특수효과 통계
- 결과 이미지 다운로드 기능

## 🎵 음악

- **레이스 중**: 업템포 배경음악
- **결과 화면**: 승리 팡파레
